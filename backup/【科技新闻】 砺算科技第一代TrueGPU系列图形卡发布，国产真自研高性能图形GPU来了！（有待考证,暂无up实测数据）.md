# 前言
>7月26日，“砥砺算芯 超越极限”砺算科技产品发布会在上海召开。
砺算首款GPU芯片“7G100”系列和首款显卡产品Lisuan eXtreme系列终于和大家见面。   
作为一款全自研高性能图形GPU，砺算7G100系列秉持创新融合，从指令集到计算核心完全由自主设计，基于自研TrueGPU天图架构，并自研指令集、自研软件栈。   
砺算7G100系列GPU以效率、平衡、拓展为重，多重性能优势达到国际主流、国内领先水平。   
比如在效率表现上，砺算7G100系列最多可同时运行48个没有依赖的任务，无需等待，能够最大化GPU使用效率，实现“智能多任务处理”。
通过打破三角形渲染的顺序限制，在不依赖顺序的场景下提升50%渲染效率，实现“智能乱序渲染”。
支持FP32或INT32的指令双发射，实现“硬件级智能分配”。
在矩阵内存布局优化层面，砺算7G100系列GPU将矩阵数据以方块形式存储，更贴近实际使用模式，优化40%显存使用效率。   
还有在平衡性上，聚焦GPU负载平衡，砺算7G100系列GPU能通过硬件自动检测与任务智能分配系统，将巨量的3D渲染和通用计算任务划分为更小的子任务，依据当前硬件负载情况动态分配给计算单元，实时平衡负载。
聚焦GPU处理器资源平衡，砺算GPU能够通过TEX/UAV指令任务动态均衡系统，自动监控并调整任务分配，智能优化数据流，消除因应用程序偏好而导致的资源不均衡。   
以及围绕拓展性，砺算GPU支持NRSS动态优化渲染画质，实现对英伟达DLSS技术和AMD FSR技术的对标。
7G100还支持SRIOV技术，最多支持16路虚拟GPU，能够为云端应用部署提供更多可能性。
强大性能下，搭载7G100系列GPU的Lisuan eXtreme系列显卡取得了良好的实测成绩。   
不论是以FireStrike、Steel Nomad为代表的Benchmark跑分成绩，还是在《黑神话 悟空》、《古墓丽影 暗影》等大型游戏上，亦或者是在渲染软件Blender、视频编辑软件剪映等专业应用上，Lisuan eXtreme系列显卡均游刃有余。
    据实测数据，Lisuan eXtreme系列显卡FireStrike跑分达26800，Steel Nomad跑分达2268，《黑神话 悟空》平均FPS＞70（分辨率：1080P 画质：高）。
    最新测试显示，7月24日正式上线的国产第二款3A大作《明末：渊虚之羽》同样能流畅运行，体现出砺算显卡性能表现、软件兼容性兼具的优势。
与此同时，在当下热门的大模型应用上，Lisuan eXtreme系列显卡可以完全满足个人AIPC对DeepSeek和文生图等模型的需求。   
目光着眼于未来，Lisuan eXtreme系列显卡除了聚焦PC游戏、专业应用、AIPC，还会在云游戏、云渲染、数字孪生、虚拟/增强现实、智能座舱、具身机器人等领域广泛应用。   
    目前，Lisuan eXtreme系列显卡包含专业版和消费级两大类，预计产品送样时间为2025年8月，启动量产时间为2025年9月，敬请期待。
    下一阶段，在游戏、专业视觉渲染、数字孪生、工业设计与仿真等领域，砺算GPU将持续释放技术能力，打造自研、可控的高性能图形产品，坚持软硬件生态构建并行。

## 砺算7G100问世：国产GPU破局之路的里程碑与挑战

在英伟达与AMD主导全球GPU市场的技术格局下，砺算科技7G100系列芯片的发布不啻为一记惊雷。这款全自研GPU从指令集到软件栈的完整自主可控，标志着中国在高端图形处理器领域迈出了实质性一步。其技术亮点直指行业痛点：

*   **效率革命：** 48任务并行处理、50%渲染效率提升的“智能乱序渲染”、40%显存优化的矩阵存储技术，突破传统顺序限制；
*   **动态平衡：** 硬件级负载均衡与资源调度系统，有效应对复杂计算场景；
*   **生态兼容：** 对标DLSS/FSR的NRSS技术、16路vGPU支持、实测流畅运行《黑神话》《明末》及大模型应用，展现强大兼容性。

**国产破局的战略价值远超产品本身：**
*   **技术自主里程碑：** TrueGPU架构证明中国具备高端GPU全链路研发能力，打破“卡脖子”困局；
*   **AI与数字经济的基石：** 为AIPC、云游戏、数字孪生、智能驾驶等前沿领域提供本土算力支撑；
*   **产业生态催化器：** 刺激国内软硬件协同创新，推动游戏引擎、工业软件等生态环节发展。

**然而，登顶之路仍荆棘密布：**
*   **生态壁垒：** CUDA生态的护城河极深，开发者迁移成本高昂，需打造差异化开发生态；
*   **持续创新压力：** 国际巨头迭代迅猛，光追、AI单元等领域需加速突破；
*   **市场信任建设：** 消费级市场对稳定性、驱动优化极为敏感，需长期投入赢得用户口碑。

**未来展望：砺算的发布是起点而非终点。** 中国GPU产业需在三个维度持续发力：
1.  **构建开放生态：** 推动跨平台开发工具链，降低开发者适配门槛；
2.  **深耕垂直场景：** 在云渲染、工业仿真等B端领域建立优势，逐步辐射消费市场；
3.  **产学研协同：** 联合高校攻关前沿架构，储备高端人才。
