# 介绍
《双影奇境》（Split Fiction）是由Hazelight Studios研发，[Electronic Arts](https://baike.baidu.com/item/Electronic%20Arts/906862?fromModule=lemma_inlink)（EA）发行的一款双人合作[冒险游戏](https://baike.baidu.com/item/%E5%86%92%E9%99%A9%E6%B8%B8%E6%88%8F/18372?fromModule=lemma_inlink)，于2025年3月7日正式发售登陆PS5、Xbox以及PC平台。
Switch 2版《双影奇境》2025年6月5日作为NS2首发作品发售。
该游戏玩家扮演风格迥异的女作家米欧和佐伊，因陷入自己创作的故事世界而展开冒险。两位素昧生平的作家被困在了对方的故事中，唯有携手合作才能逃出困境、重新回到现实，并保住自己的作品 。游戏支持本地合作、在线合作、跨平台联机，玩家可以使用好友通行证邀请一名好友免费在线游戏，只要两人其中一人购买了游戏，两人便可以共同游玩。
2025年3月，《双影奇境》获得三项吉尼斯世界纪录：1、在发售48小时内，成为销量最高的局域联机合作类电子游戏，售出超过100万份。2、在发售一周内，继续保持局域联机合作类电子游戏销量冠军的地位，总销量达到200万份。3、在Steam平台上，同时在线游玩人数最多的局域联机合作类电子游戏，峰值人数达到259003人；同月，《双影奇境》媒体评分全面解禁，并获得“必玩游戏”认证。
2025年4月，直面会公布消息，《双影奇境》将推出Switch2版本 。
