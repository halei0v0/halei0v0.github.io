# 前言
**可以获取番茄小说并下载为txt的工具**
* ✨ 功能亮点
* 图形界面与命令行双形态：新手友好、进阶高效
* 极速搜索与书籍信息展示，整本/按范围下载

* 参考并增强自 Dlmily/Tomato-Novel-Downloader-Lite，修复多个 API 逻辑并重构模块化架构。
