# 前言
**可以获取番茄小说并下载为txt的工具**
* ✨ 功能亮点
* 图形界面与命令行双形态：新手友好、进阶高效
* 极速搜索与书籍信息展示，整本/按范围下载
* 支持 TXT / EPUB 导出，封面与元数据自动处理（含 HEIC）
* 多线程下载、失败自动重试、请求限速，稳定可靠
* 自动检查更新（GitHub Releases），一键下载并重启生效
* 参考并增强自 Dlmily/Tomato-Novel-Downloader-Lite，修复多个 API 逻辑并重构模块化架构。
